## Name
ask:thing

## Synopsis
```
/ask:thing <topic>
```

## Description
Build a cognitive model for any general (non-technical) concept. Walks through 8 layers: essence, origin, mechanism (via analogy), components, comparisons, use cases, limitations, and keys to go deeper.

## Implementation
Read and follow the skill at `${CLAUDE_PLUGIN_ROOT}/skills/thing/SKILL.md`, applying it to the given topic.
