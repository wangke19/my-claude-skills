# Learning Roadmap Template

Use this structure when building the knowledge map for any topic.

## Stage 1: Foundation

- Core concepts and terminology
- Mental model: what this thing IS and why it exists
- First working example (simplest possible)

## Stage 2: Core Operations

- Syntax / CLI / workflow basics
- The 20% of features used 80% of the time
- Standard patterns and idioms

## Stage 3: Guided Practice

- Hands-on labs with step-by-step instructions
- Repetition drills on the core operations
- Building muscle memory

## Stage 4: Troubleshooting

- Common failure modes and their symptoms
- Debugging workflow
- Reading error messages

## Stage 5: Real Project

- Production-like case from start to finish
- Integrates all prior stages
- Suitable for a portfolio or real use
