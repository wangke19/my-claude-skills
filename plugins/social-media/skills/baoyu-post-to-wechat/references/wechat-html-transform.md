# WeChat HTML Compatibility Transformation

When publishing HTML that was **not** generated by `wechat-article-generator` (e.g., from external tools, hand-written HTML, or markdown converters), it must be preprocessed before passing to `operate_appmsg` / `publish-draft.py`. This file documents the required transformations.

## Transformation Rules

| Source Element | WeChat Behavior | Fix |
|---|---|---|
| `<style>` blocks | Stripped entirely by WeChat editor | Convert all CSS to inline `style=""` on each element |
| `<pre><code>` | Styling lost, renders as plain text | Replace with `<section>` + `<p style="white-space:pre-wrap">` + dark bg |
| `<div style="background:...">` | `background` filtered by `operate_appmsg` | Use `<section style="background:...;display:block">` instead |
| `<ul>/<li>` | Unreliable rendering, may show duplicate bullets | Pseudo-list: `<p style="margin:0 0 6px 0;padding-left:12px;">· item</p>` |
| `<ol>/<li>` | Same unreliability | `<p><b>1.</b> step</p>` format |
| `<a href="url">text</a>` | Not clickable in WeChat | Convert to footnote references `[1]`, collect at bottom |
| `<img src="data:image/...">` | Stripped by WeChat | Remove entirely (inline `<svg>` is preserved) |
| Copy hint `<div>` | Unnecessary in published draft | Remove |
| `<h2>` without inline style | WeChat strips `<style>` block h2 CSS | Add inline style with blue left border or bottom border |
| `<h3>` without inline style | Same as h2 | Add inline style with bold + color |
| `<blockquote>` | May lose styling via `operate_appmsg` | Use `<section>` with border-left + background inline style |
| `<table>` without inline styles | Border/padding lost | Add inline `border`, `padding`, `border-collapse` on each element |

## Code Block Pattern (Dark Background)

WeChat's `operate_appmsg` API may not properly handle `<pre><code>`. Use `<section>` wrapping instead:

```html
<section style="background:#1e1e2e;border-radius:8px;padding:16px;margin:16px 0;overflow-x:auto;">
  <p style="color:#abb2bf;font-family:'Courier New',Consolas,monospace;font-size:13px;line-height:1.7;white-space:pre-wrap;margin:0;word-break:break-all;">
    code content here
  </p>
</section>
```

Key properties:
- `white-space:pre-wrap` preserves formatting while allowing wrap
- `word-break:break-all` prevents horizontal overflow on long strings
- Dark background `#1e1e2e` gives a "code editor" feel
- Monospace font stack covers Windows/Mac/Linux

## Title Styling Pattern

```html
<!-- h2 with blue bottom border -->
<h2 style="font-size:20px;margin-top:8px;margin-bottom:12px;color:#111;border-bottom:3px solid #1a73e8;padding-bottom:8px;font-weight:700;">Section Title</h2>

<!-- h3 bold + color -->
<h3 style="font-size:17px;margin-top:24px;margin-bottom:10px;color:#333;font-weight:600;">Subsection</h3>
```

## Link → Footnote Conversion

```html
<!-- Before -->
<p>See <a href="https://example.com/docs">documentation</a> and <a href="https://github.com/repo">repo</a></p>

<!-- After -->
<p>See documentation[1] and repo[2]</p>

<!-- At bottom of article -->
<section style="margin-top:24px;padding-top:12px;border-top:1px solid #eee;font-size:13px;color:#888;">
  <p style="margin:0 0 4px 0;">[1] https://example.com/docs</p>
  <p style="margin:0 0 4px 0;">[2] https://github.com/repo</p>
</section>
```

## Blockquote → Section Conversion

```html
<!-- Before -->
<blockquote>Important note</blockquote>

<!-- After -->
<section style="border-left:4px solid #1a73e8;padding:11px 18px;margin:18px 0;background:#f0f6ff;border-radius:0 8px 8px 0;color:#444;">
  <p style="margin:0;">Important note</p>
</section>
```

## When to Use This Transform

1. **External HTML files** not generated by `wechat-article-generator`
2. **Markdown converter output** (pandoc, marked, etc.)
3. **Hand-written HTML** that uses `<style>` blocks, classes, or `<div>` decorative blocks
4. **HTML with code blocks** that need dark background styling for WeChat

Do NOT apply this transform to HTML already produced by `wechat-article-generator` — that skill handles WeChat compatibility at generation time.

## Template Script

See `templates/wechat-transform.py` for a reusable Python script that applies all these transformations automatically.
